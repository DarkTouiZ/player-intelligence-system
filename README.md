# player-intelligence-system
### machine learning challenge consists of 5 core tasks that simulate real-world gaming industry problems.
* Task 1: Anti-Cheat Pre-Filter (Binary Classification)
* Task 2: Player Segment Classification (Multi-class Classification)
* Task 3: Monthly Spending Prediction (Regression)
* Task 4: Game Title Detection (Image Classification)
* Task 5: Account Security Monitoring (Anomaly Detection)
Each task represents a critical component of the Player Intelligence System.
